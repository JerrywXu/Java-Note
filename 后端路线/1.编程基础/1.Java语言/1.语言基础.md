# 接口

在接口里面定义一个抽象方法

```java
public interface MyInterence {
    public abstract void methodAbs();//这是一个抽象方法
    //抽象方法的修饰符必须是两个固定的关键字 public abstract,或者习惯性省略
    public  void methodAbs2();
    abstract void methodAbs3();
    void  methodAbs4();
    //以上都是抽象方法
    
}

```
